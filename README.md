# Quotes App (Курсова работа по React)

## Описание
Приложение за споделяне и управление на вдъхновяващи цитати.  
Използва React и json-server (REST API). Всеки потребител може да се регистрира, да влезе, да добавя, редактира и трие свои цитати.

## Основни функции
- Регистрация с потребителско име и имейл
- Вход с проверка за съществуващ потребител
- Добавяне на нови цитати
- Преглед на всички и само на личните цитати
- Редакция и изтриване само на собствените
- Търсене по ключова дума
- Съобщения (алърти) при действия
- Визуално оформление с Bootstrap

## Технологии
- React
- React Router DOM
- Bootstrap 5
- json-server (REST API)
- Git + GitHub

## Стартиране на проекта
1. Инсталирай зависимостите:
   npm install
   
2.Стартирай json-server (в отделен терминал):
  npx json-server --watch db.json --port 5000
  
3.Стартирай React приложението:
  npm start
